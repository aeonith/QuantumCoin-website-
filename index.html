Knowledge

FAQ

Is wQC the same as QC?

No—wQC is an ERC-20 IOU used to bootstrap liquidity and awareness. At mainnet, wQC burns are mapped 1:1 to native QC delivery under a transparent redemption process.

What is RevStop™?

An owner-local, opt-in pause meant for emergencies (e.g., compromised device). It is not a custodial freeze and does not add a global killswitch.

Do I need a huge disk?

No. Run a pruned node (≈5–20 GB typical) to validate current state. Full archival nodes remain for explorers and research.

Is there any premine?

No. QC is mined via Proof-of-Work with a fixed 22M cap and a deterministic halving schedule.
